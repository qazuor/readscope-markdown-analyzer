# ReadScope

Analizador de contenido Markdown que calcula tiempo de lectura, revisa la estructura de headings y marca posibles problemas de legibilidad.

- Features
  - Cálculo de tiempo estimado de lectura basado en la cantidad de palabras
  - Resumen de estructura: niveles de headings, índice básico y densidad de secciones
  - Detección de párrafos excesivamente largos o poco escaneables

- Stack
  - React
  - TypeScript
  - Tailwind CSS
